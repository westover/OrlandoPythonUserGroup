These examples were collected for a presentation at the Orlando Python User
Group by Craig Finch on 22 May 2014. They are designed to introduce NumPy,
SciPy, and matplotlib.

The slides that accompany these examples can be found at:
https://speakerdeck.com/cfinch/introduction-to-numpy-scipy-and-matplotlib

The examples in this directory, and its subdirectories, were developed and
validated with the following software:

Python 2.7.6
NumPy 1.8.1
SciPy 0.13.3
matplotlib 1.3.3
